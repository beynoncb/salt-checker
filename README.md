
# UK Salt Intake Checker

An interactive tool to help patients estimate daily salt intake based on UK food portions. Designed for kidney health awareness.

## Features
- Colour-coded feedback (Low, Moderate, High)
- NHS-aligned tips for reducing salt
- Simple scoring algorithm based on UK salt values

## How to Use
1. Download or clone this repository.
2. Open `salt-checker.html` in any modern browser (Chrome, Edge, Firefox, Safari).
3. Select your food frequency and added salt habits.
4. Click **Calculate Intake** to see your result.
